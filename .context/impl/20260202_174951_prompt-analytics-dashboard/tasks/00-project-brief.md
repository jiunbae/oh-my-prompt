# Prompt Analytics Dashboard - Project Brief

## Overview
Design a dashboard service to analyze and visualize prompts saved from Claude Code sessions.

## Data Source
- **Storage**: MinIO (minio.example.com)
- **Content**: Claude Code conversation prompts/responses
- **Format**: TBD (need to investigate actual data structure)

## Goals
1. **Dashboard View**: Detailed view of individual prompts
2. **Analytics & Insights**:
   - Usage patterns over time
   - Token usage analysis
   - Topic/category clustering
   - Prompt effectiveness metrics
   - Common patterns and templates
3. **Search & Filter**: Find specific prompts quickly
4. **Export/Share**: Share insights or prompts

## Planning Deliverables
Each AI agent will provide perspective on:
1. Architecture design
2. Tech stack recommendations
3. Data model design
4. Key features prioritization
5. Implementation roadmap

## Constraints
- Personal project (single user initially)
- Should be cost-effective
- Modern, clean UI
